## ☕ Description


반려견과 함께 동반할 수 있는 나만의 북마크를 확인하는 웹 사이트입니다.  
내 주변 혹은 내가 북마크한 장소를 손쉽게 찾을 수 있습니다. 

강아지가 흔히 '마킹'을 한다고 하죠.
자신의 영역을 표시하는것에 영감을 두어 프로젝트 이름은 펫 마킹입니다 :)
